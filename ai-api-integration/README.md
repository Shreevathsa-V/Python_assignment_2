# AI API Integration Assignment

## Description
This project integrates multiple AI APIs including Groq, Ollama, Hugging Face, Google Gemini, and Cohere.

## Setup

1. Clone repo
2. Install dependencies:
    pip install -r requirements.txt

3. Create .env file with API keys
    - GROQ_API_KEY=your_key
    - HUGGINGFACE_API_KEY=your_key
    - GOOGLE_API_KEY=your_key
    - COHERE_API_KEY=your_key

## Run

- python groq_example.py
- python ollama_example.py
- python huggingface_example.py
- python gemini_example.py
- python cohere_example.py

## Features
- User input-based prompt system
- Error handling implemented
- Environment variable usage for security
